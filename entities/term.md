### Term

#### Fields
These are all the fields you can query on the Term entity. 
```
{
  term {
    hierarchy
    id
    parent
    summary
    term
    thesaurus
  }
}
```
#### Arguments
These are all the possible arguments for an Term query:
```
page
size

aggregations

id
summary
thesaurus
```

#### Sorting
The available sorting terms are:
```
id
summary
```
#### Aggregations
You can currently aggregate across these Term fields:
```
thesaurus
```
